бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ (бһӮбҹ’бһҡбһ”бҹӢбһңбҹҗбһҷ/бһҹбһҳбҹ’бһ”бһ‘бһ¶)бһ–бһјбһҖбҹӮ (бһӮбҹ’бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ)бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһҳбһ¶бһ“бһҹбһҳбҹ’бһ”бһ‘бһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“)бһҠбһҫбһҡбһӣбҹҒбһ„

For cardiovascular health, these are roughly equivalent:

Option A: Hike briskly (вүҘ100 spm) for 30 minutes
Option B: Run moderately for 15 minutes

Guideline: Running provides ~2Г— cardiovascular stimulus per minute
Therefore: 150 min/week hiking вүҲ 75 min/week running

  • бһҖбһјбһӣбҹҒбһҹбҹ’бһҸбҹҒбһҡбҹүбһ»бһӣбһҒбҹ’бһ–бһҹбҹӢбҹ– бҹ§.бҹ % бһ‘бһӣбҹӢбһ“бһ№бһ„ бҹӨ.бҹЈ%
